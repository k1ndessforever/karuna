> *"We don't beg for sympathy it's within us."*

An open-source, privacy-first animal welfare platform for everybody.  
Built for the voiceless. Run by the compassionate. Owned by no one.

---

What is this?

Karuna connects people who care about stray and wild animals —  
enabling real-time rescue coordination, adoption listing, feeding spot mapping,  
sterilization drives, and honest memorials for animals who didn't make it.

No donations begged. No emotions manipulated.  
Just real people. Real animals. Real help.

---

Why it exists

In villages across Maharashtra and everywhere rural part India  
an injured dog on the highway has nobody to call.  
A stray cat having kittens in the rain has no shelter.  
A cow abandoned with a wound has no vet.  
And the people who do care are working alone, exhausted, invisible.

Karuna is the connective tissue.

---

What it does

| Feature | Description |
|---|---|
| 🗺️ Map-First | All reports live on OpenStreetMap — zero Google |
| 🆘 Rescue | Report injured animals anonymously with location |
| 🏠 Adoption | List strays available, match with verified homes |
| 🍲 Feeding | Map community feeding spots, coordinate volunteers |
| ✂️ Sterilization | Organize ABC drives, connect NGOs and vets |
| 🕊️ Memorial | Honest records of animals who passed — named, remembered |
| 🔒 Anonymous | Email + OTP only. No phone. No real name. No tracking |
| 🌐 12 Languages | English, Hindi, Marathi, Urdu + 8 global languages |

---

Tech stack

```
Maps        →  Leaflet.js + OpenStreetMap  (open source, no Google)
Database    →  Supabase                    (free tier, PostgreSQL)
Auth        →  Supabase Email OTP          (no password, no phone)
AI          →  Anthropic Claude API        (fraud detection, memorials)
Hosting     →  Vercel                      (free tier)
Mobile      →  OsmAnd / Organic Maps       (F-Droid, offline capable)
```

No paid tools. No tracking scripts. No ads. Ever.

---

Privacy — the short version

✅ Collected — email (OTP only), nickname, report content, timestamp  
🚫 Never collected — phone number, real name, GPS, IP address, device info  
🚫 Never installed — Google Analytics, Meta Pixel, any ad tracker  

Full transparency at `/about-karuna.html` inside the platform.

Deploying to Vercel

